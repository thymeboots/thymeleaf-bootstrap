<h1>ThymeBoots</h1>
<p><strong>ThymeBoots</strong> is a lightweight <strong>Thymeleaf</strong> framework based on <strong>Bootstrap</strong> (for front-end) that lets you develop Front-end Enterprise Applications fast and easy</p>
