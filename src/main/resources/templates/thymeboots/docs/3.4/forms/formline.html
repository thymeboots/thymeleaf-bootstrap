FormLine

<tb:formline> component wrap control components.

This is a ThymeBoots container thats why thymeboots.css must be include

Example

Html markup
<form>
  <tb:formline>
      <label >Email address:</label>
      <input type="email" class="form-control" placeholder="Enter email">
  </tb:formline>
  <tb:formline>
      <label >Password:</label>
      <input type="password" class="form-control" placeholder="Enter password">
  </tb:formline>
