Column

Break a new column

The <tb:column> tag is a container you can put html code in.

Example

1 of 2 2 of 2
Html markup
  <tb:container>
    <tb:row>
      <tb:column>
         1 of 2
      </tb:column>
      <tb:column>
         2 of 2
      </tb:column>
    </tb:row>
  </tb:container>

Mix and match

col-12 col-md-8 col-6 col-md-4 col-6 col-md-4 col-6 col-md-4 col-6 col-md-4 col-6 col-6

      </tb:column>

         col-6 col-md-4

wd atribute ( w-%): w-100 means breakline

col1 col2 col3 col4
  <tb:container>
    <tb:row>
      <tb:column> col1 </tb:column>
      <tb:column> col2</tb:column>
      <tb:column wd="100"></tb:column>
      <tb:column> col3 </tb:column>
      <tb:column> col4 </tb:column>
    </tb:row>
  </tb:container>
