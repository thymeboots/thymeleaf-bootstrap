InputGroup

The <tb:inputgroup> component wrap one or more control components.

Example

$ .00
Html markup
<form>
  <tb:inputgroup prepend="$" append=".00" >
      <input type="text" class="form-control">
  </tb:inputgroup>
 
  <tb:inputgroup prepend="$" append=".00" look="default" >
      <input type="text" class="form-control" value="Look Default">
  </tb:inputgroup>
 
  <tb:inputgroup prepend="$" append=".00" look="default" size="sm">
      <input type="text" class="form-control" value="Look Default And Small Size">
  </tb:inputgroup>
 
  <tb:inputgroup>
      <tb:facet name="prepend"> <span class="input-group-text">$</span></tb:facet">
      <tb:facet name="append" > <span class="input-group-text">.00</span></tb:facet">
      <input type="text" class="form-control" value="Facet Example">
  </tb:inputgroup>
</form>
