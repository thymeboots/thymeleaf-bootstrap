Input

The <tb:input> control component extends from html input

Example:

Input group:

$ .00
Html markup
<tb:formgroup>
  <tb:input value="Input value">
      <tb:facet name="prepend"> <span class="input-group-text">$</span></tb:facet">
      <tb:facet name="append" > <span class="input-group-text">.00</span></tb:facet">
  </tb:input>
</tb:formgroup>

Input label:

Html markup
<tb:formgroup>
  <tb:input label="Input label"></tb:input>
</tb:formgroup>

Input radio:

Html markup
<tb:formgroup>
    <tb:input type="radio" label="Option 1" name="groupOfDefaultRadios2" checked>></tb:input>
    <tb:input type="radio" label="Option 2" name="groupOfDefaultRadios2">></tb:input>
    <tb:input type="radio" label="Option 3" name="groupOfDefaultRadios2">></tb:input>
</tb:formgroup>
 
<tb:formgroup>
    <tb:input type="radio" label="Option 1" layout="inline" name="groupOfDefaultRadios" checked>></tb:input>
    <tb:input type="radio" label="Option 2" layout="inline" name="groupOfDefaultRadios">></tb:input>
    <tb:input type="radio" label="Option 3" layout="inline" name="groupOfDefaultRadios">></tb:input>
</tb:formgroup>

Input checkbox:

Html markup
<tb:formgroup>
    <tb:input type="checkbox" label="Option 1" layout="inline" checked>></tb:input>
    <tb:input type="checkbox" label="Option 2" layout="inline" >></tb:input>
    <tb:input type="checkbox" label="Option 3" layout="inline" disabled>></tb:input>
</tb:formgroup>

Input switch:

Html markup
<tb:formgroup>
    <tb:input type="switch" label="Toggle For Option 1" checked>></tb:input>
    <tb:input type="switch" label="Toggle For Option 2" >></tb:input>
    <tb:input type="switch" label="Toggle For Option 3" disabled>></tb:input>
</tb:formgroup>

Input range:

Html markup
<tb:formgroup>
    <tb:input type="range" label="Custom range"></tb:input>
</tb:formgroup>

Input file:

Html markup
<tb:formgroup>
    <tb:input type="file" label="Custom Choose File"></tb:input>
</tb:formgroup>
 
<tb:formgroup>
    <tb:input type="file" label="Choose File" custom="false"></tb:input>
</tb:formgroup>
Attribute Default value Description
custom true true:custom component is enabled false:custom disabled
label component label
layout vertical can be:inline, vertical ( it is enabled for radio,checkbox,switch input types )
size can be lg, md, sm
type text input types:
text,number,radio,checkbox,password,date,file,range etc.
switch is a thymeboots input type , not standart input type
