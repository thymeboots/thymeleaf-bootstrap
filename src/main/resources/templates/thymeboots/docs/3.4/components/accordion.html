Accordion

The <tb:accordion> component wrap one or more <tb:card> components to display them as an accordion.

Example

Click the accordions below to expand/collapse the accordion content.

Content 1 Content 2 Content 3
Html markup
<tb:accordion expanded-index="0">
    <tb:tab header="Accordion Item #1">Content 1</tb:tab>
    <tb:tab header="Accordion Item #2">Content 2</tb:tab>
    <tb:tab header="Accordion Item #3">Content 3</tb:tab>
</tb:accordion>
Attribute Default value Description
expanded-index 0 Comma separated list of child card index that need to render expanded.
