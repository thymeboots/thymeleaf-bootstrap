The <tb:navbar> tag

Example

Search
Html markup
<tb:navbar brand="Brand" brand-href="#" fluid="true">
   <tb:navbarList>
      <tb:navLink value="Link1" href="#"></tb:navLink>
      <tb:navLink value="Link2" href="#"></tb:navLink>
      <tb:navLink value="Disabled" disabled></tb:navLink>
      <tb:dropMenu value="DropMenu">
         <tb:navLink value="Link1" href="#"></tb:navLink>
         <tb:divider></tb:divider>
         <tb:navLink value="Link2" href="#"></tb:navLink>
         <tb:navLink value="Link3" href="#"></tb:navLink>
         <tb:navLink></tb:navLink>
         <tb:navLink value="Link4" href="#"></tb:navLink>
      </tb:dropMenu>
      <tb:dropMenu value="DropMenu2">
         <tb:navLink type="dropitem" value="Link1" href="#"></tb:navLink>
         <tb:navLink type="dropitem" value="Link2" href="#"></tb:navLink>
      </tb:dropMenu>
      <tb:dropMenu value="DropMenuItem">
         <tb:dropItem value="Drop Item 1" href="#"></tb:dropItem>
         <tb:dropItem value="Drop Item 2" href="#"></tb:dropItem>
      </tb:dropMenu>
   </tb:navbarList>
   <form class="form-inline my-2 my-lg-0">
      <tb:input class="mr-sm-2" type="search" placeholder="Search" aria-label="Search"></tb:input>
      <tb:button class="my-2 my-sm-0" type="search" look="outline-success" type="submit">Search</tb:button>
   </form>
</tb:navbar>
Attribute Default value Description
brand Brand name
brand-href Brand link
fluid false true:container-fluid , false:container
fixed fixed position. [top,bottom]
sticky false true:sticky active, not fixed
sticky=true fixed=top means sticky-top
