ButtonDrop

The <tb:buttonDrop> tag is drop down button

Example

Menu Item1 Menu Item2 Menu Item1 Menu Item2 Link 1 Link 2
Html markup
<tb:buttonDrop value="Click Me For Open Drop Menu" look="primary">
   <tb:navLink href="#">Menu Item1</tb:navLink>
   <tb:navLink href="#">Menu Item2</tb:navLink>
</tb:buttonDrop>
<tb:buttonDrop value="Click Me [Small]" look="primary" size="sm">
   <tb:navLink href="#">Menu Item1</tb:navLink>
   <tb:navLink href="#">Menu Item2</tb:navLink>
</tb:buttonDrop>
<tb:buttonDrop value="Click Me [Link]" look="link" size="sm">
   <tb:navLink href="#" type="link">Link 1</tb:navLink>
   <tb:navLink href="#" type="link">Link 2</tb:navLink>
</tb:buttonDrop>
Attribute Default value Description
look secondary can be link, warning , primary , secondary, info , success , danger , error, dark , light
size can be lg, md, sm
