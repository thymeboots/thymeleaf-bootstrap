ButtonGroup

The <tb:buttonGroup> tag

Example

Html markup
<tb:buttonGroup>
    <tb:button look="primary" value="1"></tb:button>
    <tb:button look="primary" value="2"></tb:button>
</tb:buttonGroup>

Size

Html markup
<tb:buttonGroup size="lg">
    <tb:button look="primary" value="1"></tb:button>
    <tb:button look="danger" value="2"></tb:button>
</tb:buttonGroup>
Attribute Default value Description
size Button group size. can be lg, md, sm
