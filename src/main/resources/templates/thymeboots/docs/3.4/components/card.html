Card

The <tb:card> tag is a container you can put html code in

Example

Content Content Add this message to footer
Html markup
<tb:card header="Header" card-title="Card title" card-text="Card text message"></tb:card>
<tb:card header="Header" footer="Footer"> Content </tb:card>
<tb:card header="Collapsible" collapsible="true" look="dark">
   Content
   <tb:facet name="footer">
      Add this message to footer
   </tb:facet>
</tb:card>
Attribute Default value Description
card-text Card text
card-title Card title
collapsible false true means collapsible card
header Card header text
footer Card footer text
look secondary can be warning , primary , secondary, info , success , danger , error, dark , light
