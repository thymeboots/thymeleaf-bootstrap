Button

The <tb:button> tag

Example

Bootstrap includes several predefined button styles, each serving its own semantic purpose, with a few extras thrown in for more control.

Primary Secondary Success Danger Warning Info Light Dark Link
Html markup
<tb:button look="primary">Primary</tb:button>
<tb:button look="secondary">Secondary</tb:button>
<tb:button look="danger">Danger</tb:button>
<tb:button look="warning">Warning</tb:button>
<tb:button look="info">Info</tb:button>
<tb:button look="light">Light</tb:button>
<tb:button look="dark">Dark</tb:button>
<tb:button look="link">Link</tb:button>

Outline buttons

Primary Secondary Success
Html markup
<tb:button look="outline-primary">Outline Primary</tb:button>
<tb:button look="outline-secondary">Outline Secondary</tb:button>
<tb:button look="outline-success">Outline Succes</tb:button>

Sizes

Medium Small
Html markup
<tb:button look="primary" size="lg" value="Large"></tb:button>
<tb:button look="primary" size="md" value="Medium"></tb:button>
<tb:button look="primary" size="sm" icon="save"> Small</tb:button>
Attribute Default value Description
icon Font Awesome Icon Name
look secondary Look of the Button, can be warning , primary , secondary, info , success , danger , error, dark , light.
For outline add "outline-" prefix. outline-primary etc.
size can be lg, md, sm
value Button label value
