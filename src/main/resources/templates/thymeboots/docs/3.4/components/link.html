Link

The <tb:a> tag extend from HTML <a>

Example

Link 1
Html markup
<tb:a href="#">Link 1</tb:a>
<tb:a href="#" value="Link 2" look="success"></tb:a>
<tb:a href="#" value="Link 3" look="danger"></tb:a>
<tb:a href="#" value="Link 4" look="warning"></tb:a>

Button Example

Html markup
<tb:a href="#" type="button" look="primary" value="Link 1"></tb:a>
<tb:a href="#" type="button" look="success" value="Link 2"></tb:a>
<tb:a href="#" type="button" look="danger" value="Link 3" size="sm"></tb:a>

Dropitem Example

Html markup
<tb:a href="#" type="dropitem" value="Link 1"></tb:a>
<tb:a href="#" type="dropitem" value="Link 2"></tb:a>
<tb:a href="#" type="dropitem" value="Link 3"></tb:a>

Groupitem Example

Html markup
<tb:a href="#" type="groupitem" value="Link 1"></tb:a>
<tb:a href="#" type="groupitem" value="Link 2"></tb:a>
<tb:a href="#" type="groupitem" value="Link 3"></tb:a>
Attribute Default value Description
disabled false:not disabled, true or null:disabled
