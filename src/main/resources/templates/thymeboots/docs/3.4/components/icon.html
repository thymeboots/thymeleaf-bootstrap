Icon

The <tb:icon> tag uses font-awesome or glyphicon icons. Default font-awesome

Example

fa fa-info-circle
fa fa-info-calculator
Html markup
<tb:icon value="info-circle" type="fa"></tb:icon>
<tb:icon value="calculator"></tb:icon>
<tb:icon value="ok-sign" type="glyphicon" ></tb:icon>
Attribute Default value Description
type fa icon type. can be fa or glyphicon
value icon name
