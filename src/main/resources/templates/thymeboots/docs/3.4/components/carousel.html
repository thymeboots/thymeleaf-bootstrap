Carosel

The <tb:carousel> tag

Example

Carosel Item 1 Carosel item caption Carosel Item 2 Carosel Item 3
Html markup
<tb:carousel startAnimation="true" >
   <tb:carouselItem >
      Carosel Item 1
      <tb:carouselCaption >Carosel item caption</tb:carouselCaption>
   </tb:carouselItem>
   <tb:carouselItem >Carosel Item 2</tb:carouselItem>
   <tb:carouselItem >Carosel Item 3</tb:carouselItem>
</tb:carousel>
Attribute Default value Description
startAnimation true true: start animation , false:do not start
