Alert

Provide contextual feedback messages for typical user actions with the handful of available and flexible alert messages.

The <tb:alert> tag is a container you can put html code in

Example

Click the accordions below to expand/collapse the accordion content.

Success message Primary message Warning message Info message Danger message Message body
Html markup
<tb:alert severity="success">success message</tb:alert>
<tb:alert severity="primary">primary message</tb:alert>
<tb:alert severity="warning">warning message</tb:alert>
<tb:alert severity="info">info message</tb:alert>
<tb:alert severity="danger">danger message</tb:alert>
<tb:alert severity="warning" header="Header" closable="true">warning message</tb:alert>
Attribute Default value Description
closable false If true close button will be displayed.
header alert header
severity
look (alternative writing)		 warning Severity of the Alert, can be warning , primary , secondary, info , success , danger , error, dark , light
