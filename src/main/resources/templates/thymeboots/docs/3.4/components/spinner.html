Spinner

The <tb:spinner> tag

Example

Html markup
<tb:spinner type="grow" value="Loading..."></tb:spinner>
<tb:spinner type="border" value="Loading..."></tb:spinner>
<tb:spinner look="primary" size="lg"></tb:spinner>
<tb:spinner look="success" size="md"></tb:spinner>
<tb:spinner look="danger" size="sm"></tb:spinner>
Attribute Default value Description
look secondary Look of the Spinner, can be warning , primary , secondary, info , success , danger , error, dark , light
size Spinner size. can be lg, md, sm
type border Spinner type. can be grow, border
