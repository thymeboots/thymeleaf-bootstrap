ButtonModal

The <tb:buttonModal> tag is toggle the modal (open or close)

The <tb:buttonModal> is extended from <tb:button>

Example

Html markup
<tb:buttonModal target="myModal" look="primary" value="Open Modal" ></tb:buttonModal>
<tb:modal id="myModal" header="Modal1" value="Content" ></tb:modal>
Attribute Default value Description
target modal id that will be toggled
