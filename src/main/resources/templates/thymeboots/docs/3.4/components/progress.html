Progress

The <tb:progress> tag

Example










Html markup
<tb:progress value="0"></tb:progress>
<tb:progress value="20"></tb:progress>
<tb:progress value="30" look="success"></tb:progress>
<tb:progress value="40" look="info"></tb:progress>
<tb:progress value="50" look="warning"></tb:progress>
<tb:progress value="60" look="danger" label="%60 Completed"></tb:progress>
<tb:progress value="70" look="warning" striped="true"></tb:progress>
<tb:progress value="90" min="0" max="100" look="warning" animated="true"></tb:progress>
<tb:progress value="100" min="0" max="100" look="warning" animated="true" label-show="false" tooltip="Tool tip text"></tb:progress>
Attribute Default value Description
animated false true:start animation
striped false true:enable striped
look primary Look of the progress, can be primary , warning, secondary, info , success , danger , error, dark , light.
label label which will be displayed
label-show true true:show percentage as label false:dont show
min 0 minumum value
max 0 maximum value
rendered true true:render false: do not render
tooltip true The text of tooltip
tooltip-position auto where is the tooltip to be displayed? can be: top,bottom,right,left,auto
value 0 value now
