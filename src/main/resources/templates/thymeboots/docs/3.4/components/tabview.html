Tab

The <tb:tabView> component wrap one or more <tb:card> components to display them as an tab view.

Example

Content0 Content1 Content2 Content0 Content1 Content2 Content0 Content1 Content2
Html markup
<tb:tabView >
   <tb:tab header="Title0">Content0</tb:tab>
   <tb:tab header="Title1">Content1</tb:tab>
   <tb:tab header="Title2">Content2</tb:tab>
</tb:tabView>
<tb:tabView active-index="1" tab-position="bottom" >
   <tb:tab header="Title0">Content0</tb:tab>
   <tb:tab header="Title1">Content1</tb:tab>
   <tb:tab header="Title2">Content2</tb:tab>
</tb:tabView>
<tb:tabView active-index="1" tab-position="vertical" pills="true" >
   <tb:tab header="Title0">Content0</tb:tab>
   <tb:tab header="Title1">Content1</tb:tab>
   <tb:tab header="Title2">Content2</tb:tab>
</tb:tabView>
Attribute Default value Description
active-index 0 index no of the active page
tab-position tab position. can be bottom, vertical
pills false true enables tab pills
